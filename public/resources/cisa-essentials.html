CISA Cybersecurity Essentials

Six Essential Practices

1. Multi-Factor Authentication (MFA)

Require additional verification beyond passwords for all critical accounts.

2. Regular Software Updates

Keep all software current with latest security patches.

3. Email Security

Protect against phishing and email-based threats.

4. Secure Remote Access

Ensure remote connections are secure and monitored.

5. Data Backup and Recovery

Follow 3-2-1 backup rule: 3 copies, 2 media types, 1 offsite.

6. Incident Response Planning

Have documented procedures for responding to cyber incidents.
